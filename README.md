# 📊 Bellabeat Fitness Data Analysis (Power BI Dashboard)

## 🔍 Project Overview
This project is part of the Google Data Analytics Capstone. The goal is to analyze fitness tracker data and provide actionable insights for Bellabeat, a wellness company focused on smart devices for women.

The analysis focuses on user activity, sleep patterns, and sedentary behavior to support data-driven business decisions.

---

## 🎯 Business Objective
To identify trends in user behavior and provide recommendations that can help Bellabeat:
- Improve user engagement
- Promote healthier lifestyles
- Enhance marketing strategies

---

## 📁 Dataset
- Source: Fitbit Fitness Tracker Data (public dataset)
- Data includes:
  - Daily activity (steps, calories, intensity)
  - Sleep records
  - Heart rate data

---

## 🛠️ Tools & Technologies
- Power BI (Dashboard creation)
- Excel / SQL (Data cleaning & preprocessing)
- Data Visualization
- Data Analysis

---

## 📊 Dashboard Features

### 1. Overview Page
- Total Steps, Calories Burned, Active Minutes
- Average Sleep Duration
- Daily Steps Trend
- Calories Burned Trend
- Sleep Breakdown
- Weekly Activity Analysis

### 2. Insights Page
- Activity Level vs Sleep Quality
- Sedentary Time vs Activity Level
- Heart Rate vs Activity
- Interactive filters (Date, Day)

---

## 🔍 Key Insights
- Users have high sedentary time (~900+ minutes/day)
- Average daily steps (~7,600) are below recommended 10,000 steps
- Higher activity levels are linked to better sleep quality
- Users are more active on weekdays compared to weekends

- Encourage users to achieve daily step goals
- Promote sleep improvement through activity tracking
